Project Requirement:

-database MySQL
-Apache Tomcat 6.0.26
-Apache Maven 3.0.4
-Eclipse Kepler

1. Before run this project import file db.sql for database.
2. Checkout and then import project ScrapperTest on eclipse.
3. Before run, on command prompt execute: mvn eclipse:eclipse -Dwtpversion=2.0
4. Run project on the server.
5. Open browser and go to http://localhost:8080/ScrapperTest/
